# IP Guard - OPC 出海开发者的 IP 风险检测工具

**IP Guard** 专为**出海开发者**设计，在项目开发阶段自动检测字体版权、许可证合规、隐私政策等知识产权风险，让 AI 编程助手帮你把关。

---

## 🎯 解决什么问题

- ⚠️ 用的字体能免费商用吗？
- ⚠️ 开源协议允许闭源/商业使用吗？
- ⚠️ 目标市场的隐私合规过关吗？
- ⚠️ 部署海外市场有哪些法律风险？

**IP Guard 是一个面向 AI 的 Skills+CLI 程序，0 额外精力+低成本就能避免侵权发生！**


---

## ✨ 核心功能

### 🔍 四大检测模块（全本地并行）

| 模块 | 检测内容 | 风险等级 |
|-----|---------|---------|
| **字体检测** | 微软雅黑、方正等商业字体，替换为开源替代方案 | 🔴高/🟡中/🟢低 |
| **许可证检测** | GPL/MIT/Apache 等开源协议分析 | 🔴高/🟡中/🟢低 |
| **隐私检测** | GDPR/CCPA/LGPD/POPIA 合规检查 | 🔴高/🟡中/🟢低 |
| **图标检测** | 图片版权风险预警 | 低 |

**四大检测模块全本地并行执行，扫描高效稳定**

### 🌍 目标市场

项目自动检测目标市场，适配各国法律框架：

| 市场 | 参数值 | 适用法规 |
|-----|--------|---------|
| 欧洲 | `--region europe` | GDPR + CCPA |
| 北美 | `--region north-america` | CCPA + GDPR |
| 东亚 | `--region east-asia` | GDPR |
| 南美 | `--region south-america` | LGPD |
| 非洲 | `--region africa` | POPIA |

## 🚀 快速开始

### 构建

```bash
pnpm install
pnpm run build
```

### 使用


构建cli与skills后，直接安装到ai编程工具使用即可，在开发收尾阶段ai会自动调用工具检查。

### 隐私检测覆盖地区

| 市场 | 法规 | 说明 |
|------|------|------|
| 欧洲 | GDPR/CCPA | 通用数据保护条例 + 加州消费者隐私法 |
| 北美 | CCPA/GDPR | 加州消费者隐私法 + 通用数据保护条例 |
| 东亚 | GDPR | 通用数据保护条例 |
| 南美 | LGPD | 巴西通用数据保护法 |
| 非洲 | POPIA | 个人信息保护法 |

## 🤖 AI 编程助手集成

在 Claude Code、Cursor、Trae 等 AI 编程助手中，IP Guard 已作为 Skill 注册。当检测到出海合规需求时，自动触发扫描并返回修复建议。
您可以选择主动告诉 Claude 检测或者让 Claude 自行决定什么时候检测


## 为什么检测准确？

本项目通过扫描前端文件+版权黑名单避雷+全本地并行检测 实现真正实用稳定的检测效果

**核心优势：**
- ✅ 全本地运行，无需 API 调用
- ✅ 四项检测并行执行，扫描高效
- ✅ 异步文件读取，稳定可靠
- ✅ 正则表达式缓存，避免重复编译


**使用示例：**
```
用户："项目要上欧洲，帮我检测 IP 风险"
   ↓
AI 执行：ipguard scan . --region europe
   ↓
AI 反馈给用户修复建议
```
